Transferência de Arquivos

Protocolo customizado em Java com TCP, JSON e validação de hash

Descrição

Sistema de transferência de arquivos via TCP, com protocolo customizado baseado em comandos LIST, PUT e GET. Implementa suporte a Base64 e verificação por hash.

Funcionalidades

  • Listagem de arquivos disponíveis no servidor.
  • Envio (PUT) e download (GET) de arquivos.
  • Validação de integridade via hash MD5.

Tecnologias

Linguagem

Java

Protocolo

TCP Sockets

Formato

JSON

Segurança

Base64, Hash MD5
