Sistema de Gerenciamento de Entregas

Aplicação Java para cálculo automático de fretes

Descrição

Sistema desenvolvido em Java com Programação Orientada a Objetos, permitindo cadastro de clientes, motoristas e veículos. O sistema realiza automaticamente o cálculo do frete com base na distância e valor por km.

Funcionalidades

  • Cadastro de clientes, motoristas e veículos.
  • Cálculo automático de frete (distância × valor/km).
  • Listagem de entregas e validação de dados.
