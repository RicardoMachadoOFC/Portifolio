Sistema de Vendas MDI

Sistema MDI (Multiple Document Interface) desenvolvido em Java com arquitetura MVC, integração com MySQL, dois perfis de usuário (Administrador e Padrão) e operações CRUD completas para categorias, produtos, clientes e vendas.

Objetivo do Projeto

Desenvolver um sistema MDI (Multiple Document Interface) em Java com:

  • Interface gráfica funcional
  • Integração com banco de dados relacional (MySQL)
  • Suporte a 2 perfis de usuário (Administrador e Padrão)
  • Operações CRUD completas em 3+ telas
  • Geração de relatórios com filtragem de dados
  • Tela de login integrada
  • Tratamento de exceções com feedback
  • Arquitetura MVC

Funcionalidades Implementadas

Sistema Principal

  • Autenticação de usuários com perfis (Admin/Padrão)
  • Dashboard MDI com navegação integrada
  • Tratamento de exceções com feedback visual

Estrutura do Projeto

Pacote Model

  • Entidades: Categoria, Cliente, Produto, Usuario, Venda, VendaDetalhes
  • Conexão BD: Conexao.java, ConexaoMysql.java
  • DAOs: AutenticacaoDao, CategoriaDao, ClienteDao, ProdutoDao, UsuarioDao, VendaDao

Pacote View

  • Telas: LoginForm, Dashboard (MDI Principal)
  • CRUDs: CategoriaView, ProdutoView, ClienteView
  • Diálogos: dialogUsuario, dialogVenda

Pacote Controller

  • Controladores: AutenticacaoController, CategoriaController, ClienteController, ProdutoController, UsuarioController, VendaController

Pacote Util

  • LoginDto.java, NegocioException.java
  • TableModels: CategoriaTableModel, ClienteTableModel, ProductoTableModel
