Sistema de Vendas

Gestão completa de vendas em Java com integração a banco de dados

Descrição

Projeto desenvolvido em Java com arquitetura MVC, interface gráfica JavaFX/Swing, operações CRUD e integração ao banco de dados MySQL.

Funcionalidades

  • Login de vendedor para cadastrar produtos, usuários e vendas.
  • Login de cliente para visualizar produtos disponíveis.
  • Controle de estoque e geração de relatórios básicos.

Tecnologias

Java, JavaFX, Swing, MySQL, POO, MVC
